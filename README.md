# ❓ QnAApp – ASP.NET Core Q&A Web App

A simple and effective Question & Answer web application built with **ASP.NET Core 8** and **Entity Framework Core**.

---

## 🧰 Technologies Used

- ⚙️ ASP.NET Core 8  
- 🗃️ Entity Framework Core  
- 🛢️ SQL Server  
- ✅ FluentValidation  

---

```bash
📦 QnAApp
